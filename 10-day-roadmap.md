## üí° **EXAM STRATEGIES**

### **General Exam Tips**

**Before the Exam:**
- Get 8 hours sleep
- Eat light breakfast
- Arrive 15 min early (or test setup if online)
- Have water available
- Use restroom before starting

**During the Exam:**
- Read questions carefully (look for keywords)
- Eliminate obviously wrong answers first
- Flag difficult questions, return later
- Don't spend >2 min per question initially
- Review flagged questions at end
- Use all available time

**Common Traps:**
- Questions with "most cost-effective" (look for cheapest option)
- Questions with "least operational overhead" (look for managed services)
- Questions with "highest availability" (look for Multi-AZ, multi-region)
- Beware of "always" and "never" in answers

---

### **AWS SAA Specific Tips**

**Question Patterns:**
1. **Scenario questions** - Read carefully, identify key requirements
2. **Service selection** - Choose based on use case (e.g., Aurora for high performance DB)
3. **Cost optimization** - S3 Glacier, Reserved Instances, Spot Instances
4. **Security** - Principle of least privilege, encryption at rest/transit
5. **High availability** - Multi-AZ, multiple regions, ELB + Auto Scaling

**Key Decision Factors:**
- Cost-effective? ‚Üí S3 Standard-IA, Reserved Instances, Lambda
- High performance? ‚Üí ElastiCache, Aurora, Provisioned IOPS
- Serverless? ‚Üí Lambda, DynamoDB, S3
- Managed? ‚Üí RDS, ElastiCache, Elastic Beanstalk
- Scalable? ‚Üí Auto Scaling, DynamoDB, S3

---

### **Azure AZ-104 Specific Tips**

**Question Patterns:**
1. **Case studies** - Read the scenario first, then questions
2. **Drag-and-drop** - Match services to requirements
3. **PowerShell/CLI commands** - Basic syntax knowledge needed
4. **RBAC scenarios** - Understand role assignments and scope

**Key Decision Factors:**
- Need identity management? ‚Üí Entra ID
- Need file sharing? ‚Üí Azure Files
- Need web hosting? ‚Üí App Service
- Need disaster recovery? ‚Üí Site Recovery
- Need monitoring? ‚Üí Azure Monitor
